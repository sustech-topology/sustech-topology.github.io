Mar 26, '26, Yiran Chen, Topology-aware Knowledge Distillation in Object Detection

In this talk, we explore the use of Topological Data Analysis in object detection, with a particular focus on separating adjacent objects that may otherwise be mistaken as a single target. The main idea is to transfer the teacher model’s ability, learned from clean data, to distinguish connected objects into a student model trained on noisy data. To achieve this, we design a specialized loss function that helps the student learn this separation capability during knowledge distillation.

Apr 16, '26, Xiaolong Han, Exploring Fermat Distance, Ideas and Early Experiments

  • Coming from physics, Fermat distance is used to model the principle that a path tends to minimize travel “cost” rather than ordinary geometric length, just as light travels along the path of least time instead of the shortest Euclidean route. In this talk I will briefly introduce the intuitive idea of Fermat distance and some attempts I made with it.
